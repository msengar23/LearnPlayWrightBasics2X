# Chapter 03: Identifiers & Literals

Comprehensive coverage of JavaScript identifiers, naming conventions, keywords, and literals with VS Code integration.

## Files

| File | Description |
|:---|:---|
| `06_Identifier_Rules.js` | Valid identifier patterns and rules |
| `07_Identifier_Literals_Part02.js` | Naming conventions deep dive |
| `08_Comments.js` | Single-line, multi-line, and JSDoc comments |
| `Identifier_Rules.md` | Reference guide for identifier rules |
| `JavaScript_Keywords.md` | List of reserved keywords |
| `JavaScript_Keywords_VS_Code.md` | Keywords with VS Code tips |
| `JavaScript_Literals_VS_Code.md` | Literals with VS Code tips |
| `VS_Code_KeyboardShortcuts.md` | Complete Windows shortcuts reference |

## Topics Covered

- **Identifier Rules** — Valid patterns, naming conventions (camelCase, PascalCase, snake_case, SCREAMING_SNAKE_CASE), reserved keywords, case-sensitivity, Unicode support
- **Naming Conventions** — camelCase, PascalCase, snake_case, SCREAMING_SNAKE_CASE, Hungarian notation
- **Comments** — `//`, `/* */`, JSDoc, VS Code shortcuts
- **JavaScript Keywords** — Declaration, control flow, loop, object/class, type checking, literals, async, module keywords
- **JavaScript Literals** — String, number, boolean, null, undefined, array, object, regex, function, Symbol, spread/rest, destructuring
- **VS Code Keyboard Shortcuts** — General, editing, navigation, search, multi-cursor, debugging, terminal shortcuts

## How to Run

```bash
node 06_Identifier_Rules.js
node 07_Identifier_Literals_Part02.js
node 08_Comments.js
```
