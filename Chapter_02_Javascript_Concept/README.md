# Chapter 02: JavaScript Concepts

This chapter provides a deep dive into core JavaScript programming concepts essential for test automation.

## Files

| File | Description |
|:---|:---|

## Topics Covered

- Variable scoping (global vs function vs block)
- Hoisting behavior
- Type coercion and truthy/falsy values
- Comparison operators (`==` vs `===`)
- Template literals and string interpolation
- Arrow functions and function expressions
- Array and object destructuring
- Spread and rest operators

node 05_JS_Basics.js
