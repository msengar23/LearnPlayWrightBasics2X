# Chapter 12: Functions

Master JavaScript functions — from basic declarations to arrow functions, parameters, return values, and function expressions.

## Files

| File | Description |
|:---|:---|
| `96_Functions.js` | Function basics — define and call |
| `97_Type1_Basic_Functions.js` | Type 1: No parameter, no return |
| `98_Type2_fn_with_Param_No_return.js` | Type 2: With parameter, no return |
| `99_Type3_Fn_without_Param_With_Return_Type.js` | Type 3: No parameter, with return |
| `100_Type4_Fn_With_Param_Without_Return.js` | Type 4: With parameter, with return |
| `101_Template_literals.js` | Template literals review |
| `102_fn_Expressions.js` | Function expressions |
| `Arrow_fn.js` | Arrow functions (ES6) |

## Topics Covered

- Function declaration and calling
- Function parameters and arguments
- Return values
- Four types of functions:
  1. No parameter, no return
  2. With parameter, no return
  3. No parameter, with return
  4. With parameter, with return
- Function expressions
- Arrow functions (`=>`) and concise syntax
- Template literals with functions
