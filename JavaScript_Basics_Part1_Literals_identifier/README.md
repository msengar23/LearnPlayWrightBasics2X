# JavaScript Basics - Part 1: Literals & Identifiers

This folder contains practice exercises and code snippets covering JavaScript literals and identifiers.

## Contents

- **Q1.js to Q15.js**: Question-based exercise files for practicing JavaScript fundamentals related to literals and identifiers.

## Topics Covered

- JavaScript Literals (string, number, boolean, object, array, etc.)
- Identifiers and Naming Conventions
- Valid vs Invalid Identifier Names
- Best Practices for Variable Naming

## How to Use

1. Open each `Q*.js` file to view the exercise.
2. Write your JavaScript code directly in the file.
3. Run the file using Node.js:
   ```bash
   node Q1.js
   ```

## Prerequisites

- Node.js installed on your machine.
- Basic understanding of JavaScript syntax.

Happy Coding!
